Portfolio

Audio Engineer Website

I designed and built a custom website for my audio engineer friend, using Bootstrap to ensure a responsive layout. I ultilize the Spotify API to keep the site updated with his discography playlist. This was my first time working with someone else to make a web project.

Fictional WordPress Business

I designed and developed a fictional antique store website using WordPress with the Astra theme, exploring its design capabilities to create an elegant, user-friendly layout. The project initially featured WooCommerce functionality, but I streamlined the build for simplicity and hosting efficiency. To optimize performance and security, I converted the final version into a fully static HTML site using the Static Plugin.
